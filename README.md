Proyecto de Python desarrollado como ejercicio grupal para practicar el uso de **listas**, **diccionarios** y **estructuras condicionales**.

## 📌 Descripción

El programa gestiona un registro de personas con sus edades. Permite al usuario:

1. **Ver** la lista de personas registradas.
2. **Buscar** una persona por nombre.
3. **Consultar** la edad de la persona encontrada.
4. **Actualizar** la edad si el usuario lo desea.

---

## ⚙️ Funcionamiento

```
Personas registradas:
- Gian
- Adriana
- David
- ...

Escribe un nombre: Gian
Gian tiene 30 años.

¿Deseas cambiar la edad? (si/no): si
Nueva edad: 31
Edad de Gian actualizada a 31.

Hay 11 personas en el registro.
```

---

## 🧠 Conceptos aplicados

- **Listas** → almacenamiento de nombres de personas.
- **Diccionarios** → asociación de nombres con edades (`clave: valor`).
- **Condicionales (`if / else`)** → validación de búsqueda y actualización de datos.
- **Entrada del usuario (`input()`)** → interacción dinámica con el programa.
- **F-strings** → formato de mensajes de salida.
- **`.capitalize()` y `.lower()`** → normalización de texto ingresado.
